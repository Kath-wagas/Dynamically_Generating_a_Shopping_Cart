Dynamically Generating a Shopping Cart

This project is a simple e-commerce shopping cart application developed as part of the Web Systems and Technologies course. It demonstrates the use of core JavaScript concepts and DOM manipulation to dynamically add products to a shopping cart and update its contents in real time.

The application allows users to:

-Add items to the shopping cart

-View selected items instantly on the webpage

-See dynamic updates to the cart without page reloads


This activity helps strengthen understanding of:

-JavaScript fundamentals

-Event handling

-DOM manipulation

-Dynamic content generation


This project serves as a practical introduction to building interactive web applications using HTML, CSS, and JavaScript.
